#### RUN ON VPS/UBUNTU/WINDOWS

1. Install NodeJs,ffmpeg
2. Installation
   ```
   npm i -global pm2
   git clone https://github.com/Donnakuks/Dark-crash
   npm install
   ```
3. Configuration
   ```
   echo "VPS = true
     SESSION_ID = null
     AUTH_FILE = session
     SUDO = null
     PREFIX = .
     MODE = private
     LOG_MSG = true" > config.env
   ```
- Start
  ```
  npm start
  ```
- Stop
  ```
  pm2 delete DARK-CRASH
  
  ```
